Metrics collection: counters, gauges, histograms.

## When to Use

| Use Case | Recommended |
|---------|-------------|

| Alerting | ✅ |
| Dashboards | ✅ |
| Development | Minimal |

## Metric Types

| Type | Description | Example |
|------|-------------|----------|
| **Counter** | Always increasing | Total requests |
| **Gauge** | Point in time | Memory usage |
| **Histogram** | Distribution | Request latency |
