## Terms Introduced

| Term | Definition | First Use |
|------|------------|-----------|
| generate command | CLI command that scaffolds phase documentation files from the active template | Scope 01 |
| review command | CLI command that analyzes documentation quality for a given phase | Scope 02 |
| quality score | Aggregated metric (0–100) reflecting documentation completeness across all phases | Scope 05 |
| traceability matrix | Cross-phase mapping showing which domain entities and requirements are referenced in which documents | Scope 03 |
| template diff | Comparison of two template versions showing added, removed, and changed sections | Scope 04 |
| CI mode | Non-interactive execution mode that exits with a machine-readable status code | Scope 06 |
