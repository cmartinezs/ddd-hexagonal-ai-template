## Step File Structure (per file)

Each step file must contain:
1. **Goal** — what the user produces in this phase
2. **Prerequisites** — what must be done before this step
3. **Instructions** — numbered, actionable tasks (fill this section, run this check, etc.)
4. **AI Prompt** — copy-paste ready prompt for generating the phase document
5. **Done Check** — verifiable checklist the user validates before moving on
6. **Next Step** — link to next step file
