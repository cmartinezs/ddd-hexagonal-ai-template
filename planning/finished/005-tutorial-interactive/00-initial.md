## Intent

Create a self-contained interactive tutorial in `00-guides-and-instructions/tutorial/` where a user follows structured exercises to learn the documentation system, with one step file per SDLC phase, AI-ready prompts, and a validation checklist per step.

---

## Why

The existing guides (`AI-WORKFLOW-GUIDE.md`, `INSTRUCTIONS-FOR-AI.md`) explain *how* to use the system conceptually, but there is no hands-on "do this, then this, then check this" tutorial that a new user can follow from zero. An interactive tutorial reduces the onboarding barrier significantly by making the workflow experiential rather than only explanatory.

**Differentiator from 003 (full-cycle tutorial):**
- 003 = "here is what a completed project looks like" (reference)
- 005 = "now you do it, step by step" (exercise / guided practice)
